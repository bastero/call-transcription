# CallScribe Documentation Images

This directory contains screenshots and images used in the documentation.

## Current Images

### callscribe-web-ui.png
Screenshot of the CallScribe web interface showing:
- Header with CALLSCRIBE logo and meeting background
- Status bar (Idle, 0:00, Standard mode)
- Recording mode selector (Standard, Streaming, Pausable)
- Start/Pause/Stop control buttons
- Options panel with:
  - Speaker Detection toggle
  - Timestamps toggle (enabled)
  - Full Report toggle
  - Save Audio toggle
  - Speakers dropdown (2 Speakers)
  - Model dropdown (Base)
  - Language dropdown (Auto-detect)
- Live Transcript window
- Activity Log with system messages
- Mobile Access section with QR code and IP address

## Adding Images

To add the web UI screenshot:

1. Take a screenshot of the CallScribe web interface (localhost:8080)
2. Save it as `callscribe-web-ui.png` in this directory
3. The image is already referenced in:
   - `README.md` (main project documentation)
   - `WEB_GUI_GUIDE.md` (web interface guide)

## Image Guidelines

- Use PNG format for screenshots
- Keep file sizes reasonable (compress if > 1MB)
- Use descriptive filenames
- Update this README when adding new images
- Show the actual interface, not mockups
